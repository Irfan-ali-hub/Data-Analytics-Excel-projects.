# Data Analytics with Excel  

## Overview  
This repository contains my portfolio projects in **Data Analytics using Microsoft Excel**. Each project demonstrates how I collect, clean, analyze, and visualize data to extract meaningful insights. The aim is to showcase practical skills in turning raw datasets into interactive dashboards and reports that support decision-making across different domains such as education, sales, business, and more.  

---

## Project Highlights  
Each project includes:  
- Data Cleaning & Preparation → Handling missing values, duplicates, and formatting.  
- Data Analysis → Using Excel formulas, PivotTables, and advanced functions.  
- Data Visualization → Building interactive dashboards, charts, and reports.  
- Insights & Recommendations → Extracting patterns and presenting findings clearly.  

---

## Tools & Skills Used  
- Microsoft Excel (PivotTables, VLOOKUP, Index-Match, Conditional Formatting)  
- Data Cleaning and Transformation  
- Dashboard Creation and Visualization  
- Business Intelligence & Insights  

---

## Outcomes  
Through these projects, I aim to:  
- Demonstrate end-to-end data analytics workflows in Excel.  
- Show how raw data can be turned into meaningful insights.  
- Build a portfolio of real-world case studies in multiple domains.
